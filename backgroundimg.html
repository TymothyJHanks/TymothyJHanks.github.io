You're in luck, there's an amazing little web app that does just this, with any kind of image:

http://damienclarke.me/code/mosaic-maker

You can see how it works, here:

https://www.youtube.com/watch?v=vO5AbU0O-P8

Create an image with the kind of colouring you want, then use the above app. Watch the video first, you'll see how it works, and then know what to create to get the gradient you want.
